## # 前言 ##

  这是一个简洁的个人博客项目，有强大的后台管理系统，有简洁漂亮的前台文章展示系统，响应式前台系统。对于博客项目来说，这属于功能比较强大的系统了。程序这里前端

  <br />

  项目演示：[ipso的独立博客](http://www.ipso.live)
